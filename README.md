# BIOL5570 Morphology Module 2026
This is the repository for the CT scans, color codes, and morphometrics that we will be using through out this semester. 

## In this repository you will find several things:
1. The microCT scans for each of your specimens
2. The color codes sheet that will apply to all the bones we use in this class
3. The list of points (including their justification) used to answer your questions

### To complete this project we will be using a software called 3D Slicer and utilizing the package SlicerMorph.
